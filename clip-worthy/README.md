# clip-worthy

Input folder for the highlight clipping agent.

**Required files:**
- `*.mp4` or `*.webm` — A 2-minute video file
- `*.json` — Analysis data with `transcript`, `chat_activity`, and `streamer_emotions`

**Sample:** `sample_data.json` is provided for testing. Add a matching video file (e.g. `sample.mp4`) to run the clipper.
